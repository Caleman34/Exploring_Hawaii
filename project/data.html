Data & Resources

Raw Data Sets, website links, and Other info

Website Pictures and Information

  • Throughout the website, beautiful and vibrant images were used from Go Hawaii to showcase Hawaii's beauty. Descriptions of the islands and of island life were also used from Go Hawaii.

JavaScript Libraries

  • As the user scrolls, various elements of the website become animated with the help of ScrollReveal.

  • ScrollReveal is a JavaScript library for easily animating elements as they enter/leave the viewport. It was designed to be robust and flexible, and is surprisingly easy it to add to your code.

  • Another JS library used that works with scrollReveal is AniJS. AniJS provides more varieties of animation features for selected elements in a web page that add fun and excitement to your code.

Beach Conditions and Beach Information

  • Hawaii Beach Safety API is used for current beach information in the leaflet map showing the different beach locations in Hawaii

  • The live API connection was not consistent and a CORS error occured periodically. To avoid the CORS error, the .json code is copied into a .json file to use for this project. The API link is in the code, however it is commented out.

Lodging Information

  • Lodging information was downloaded from Hawaii geoportal as a .geojson for hotels throughout the Hawaiian Islands and incorporated into a leaflet map as a layer with lodging information provided in tooltips.

Hawaii Precipitation and Temperature Data

  • Climate data was copied into and excel file from Weather.gov then cleaned with pandas using python in VSC then exported into an Excel file then connected to Tableau to create an interactive dashboard.

UFO Sightings Data

  • UFO sightings data was pulled from the National UFO Reporting Center website, copied to an excel file, then converted to a js file using a CSV to json converter.

Volcano Locations

  • Volcano names and locations data were copied from NOAA into an excel spreadsheet, converted into a .json file then added as a layer to the Leaflet map.

Park Locations

  • The park polygons were downloaded as a geojson file from Hawaii geoportal and added as a leaflet layer.

Park Locations

  • The coral data was downloaded as a geojson file from NOAA and added as a leaflet layer.
